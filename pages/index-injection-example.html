Index Injection Example

Injecting small snippets via a safe script block:

<script>
  (function(){
    const el = document.querySelector('#some-target');
    if (!el) return;
    el.insertAdjacentHTML('beforeend', '<div class="notice">Hello from injection!</div>');
  })();
</script>

Keep injections small and avoid remote eval. Prefer progressive enhancement.
